# Fund Dashboard

A comprehensive dashboard for feeder fund monitoring with attribution analysis.

## Features
- Monthly subscription tracking by distributor
- Performance metrics (YTD, ITD, 3M)
- Underlying USD asset price movements
- FX rate impact analysis
- Attribution breakdown
- Internal team sharing capabilities

## Tech Stack
- Frontend: React + TypeScript + Chart.js
- Backend: Node.js + Express
- Database: PostgreSQL (with sample SQLite for development)
- Authentication: Basic auth for internal use
